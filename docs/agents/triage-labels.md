# Triage Labels

| Skill role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | More reporter information is required |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an implementation agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation or judgment |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a canonical role, use the matching GitHub label from this table.
