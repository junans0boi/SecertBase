# 사용자 노출명 '원카드' 전환, 내부 식별자는 `uno` 유지

Status: accepted (2026-07-18)

UNO는 Mattel의 상표이고 카드 외형(trade dress)도 법적 보호 대상이라, 첫 릴리즈 전에 사용자에게 보이는 모든 명칭을 **'원카드 (One Card)'**로 바꾸고 카드 아트워크(특히 +4 부채꼴 스타일, 빨/노/초/파 원색 + 타원 로고 조합)를 Secret Base 자체 디자인으로 교체한다. 게임 규칙(+2/+4 등 메커니즘)은 저작권 보호 대상이 아니므로 유지한다.

내부 식별자 — `game:uno:*` 소켓 이벤트, `uno-engine.js`, `uno_screen.dart` 등 파일명, Redis 키 — 는 사용자에게 노출되지 않아 법적 리스크가 없고, 리네임하면 소켓 프로토콜·재접속 복원(`session:restore`)·Redis 상태 전반에 회귀 리스크만 생기므로 **의도적으로 `uno`를 유지**한다. 코드가 `uno`인데 UI가 '원카드'인 것은 실수가 아니라 이 결정의 결과다.

## Considered Options

- UI 문자열만 변경 (카드 외형 유지) — 이름만 바꾼 모방작이라 우회 목적 미달성. 기각.
- 내부 식별자까지 전면 리네임 — 법적 이득 없이 회귀 리스크만 증가. 기각.
